# Data

This folder contains documentation about the dataset used in this project.

## Dataset Overview

- Country: Morocco
- Sector: Commercial Banking
- Sample: 5 banks
- Period: 2013–2022
- Panel Type: Balanced Panel

## Panel Structure

| Item | Value |
|------|------|
| Banks | 5 |
| Years | 10 |
| Observations | 50 |
| Variables | 7 |
| Data Values | 350 |


## Variables

- ROA – Return on Assets
- ADC – Capital Adequacy Ratio
- CNE – Cost-to-Income Ratio
- RCNP – Non-Performing Loan Ratio
- TB – Bank Size
- TC – GDP Growth Rate
- TMIC – Maximum Conventional Interest Rate

## Data Sources

- Annual reports of the selected banks
- Bank Al-Maghrib
- World Bank

## Confidentiality

The original dataset is not publicly available.

The repository contains only the analysis code, documentation, visualizations, and research outputs.
